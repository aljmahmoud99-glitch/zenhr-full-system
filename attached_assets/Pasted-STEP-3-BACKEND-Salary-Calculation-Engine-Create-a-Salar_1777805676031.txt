STEP 3 — BACKEND: Salary Calculation Engine
═══════════════════════════════════
Create a SalaryCalculationService (single file, pure calculation logic, no DB access):

  calculateComponentValue(component, overrideValue, basicSalary, currentEarnings):
    fixed:      return overrideValue ?? component.defaultValue
    percentage: return (overrideValue ?? component.defaultValue) / 100 × (percentageBase === 'basic' ? basicSalary : currentEarnings)
    formula:    return evaluateSafeFormula(component.formulaExpression, { basic: basicSalary, gross: currentEarnings })

  evaluateSafeFormula(expression, variables):
    ALLOWED: decimal numbers, +, -, *, /, (, ), variable names: basic, gross, hours, rate
    FORBIDDEN: everything else — validate with whitelist regex before evaluating
    Use a simple recursive descent parser — NOT eval(), NOT Function(), NOT DataTable.Compute() (no .NET here)
    If expression fails validation: return 0 and log warning
    Node.js implementation: replace variable names with values, then parse arithmetic safely

  calculateGross(employeeComponents):
    Process earnings in sort_order ASC
    For each earning component: add calculated value to running total
    Return total earnings

  calculateDeductions(employeeComponents, gross, basic, config):
    SSC:
      insurable = MIN(basic, config.ssc_max_insurable_salary)
      ssc_employee = isSSCExempt ? 0 : ROUND(insurable × config.ssc_employee_rate / 100, 3)
      ssc_employer = isSSCExempt ? 0 : ROUND(insurable × config.ssc_employer_rate / 100, 3)
    
    Income Tax (Jordanian progressive brackets):
      annual_gross = gross × 12
      exemptions = config.income_tax_personal_exemption + (hasFamily ? config.income_tax_family_exemption : 0) + employee.taxExemptionAmount
      taxable_annual = MAX(0, annual_gross - exemptions)
      annual_tax = applyBrackets(taxable_annual, config.income_tax_brackets)
      tax_monthly = ROUND(annual_tax / 12, 3)
    
    Component deductions (loans, penalty, etc.):
      For each deduction component in sort_order: calculate and add
    
    Return: { ssc_employee, ssc_employer, income_tax, component_deductions, total }

  applyBrackets(taxableAmount, brackets):
    brackets = sorted ascending by from_amount
    Apply progressively, return total tax
    Tax brackets (from system_configurations, not hardcoded):
      0–5000: 0%, 5001–10000: 5%, 10001–15000: 10%, 15001–20000: 15%, 20001+: 20%

Create PayrollRunService (handles the full run for all employees):
  For each active employee in company:
    1. Load active salary components (effective_to IS NULL OR >= run_date)
    2. Load approved overtime for this month with compensation_type='payment'
    3. Load config values (SSC rates, tax brackets, working days)
    4. Calculate gross using SalaryCalculationService
    5. Calculate deductions
    6. Create payslip with SNAPSHOT of all values (copy, not reference)
    7. Link approved OT records to payslip (set linked_payslip_id)
    8. Link approved undeducted advances to payslip (set deducted_in_run_id)
  
  Store in payroll_runs: total_gross, total_net, total_deductions, total_ssc_employee, total_ssc_employer, total_income_tax, employee_count
  Payroll run status: draft → approved → paid
